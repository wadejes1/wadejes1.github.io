Read in Data from census

I am reading in the data from the census. It gives the count for the the year and data. This particular dataset is set for April 2023, the most recent release of data. The next permits release is on July 16, which will be June permit data at the county level.

data_1 %>% 
  left_join(FipsToState,by=c("FIPSState"="FIPS_Code"))%>% 
  mutate(CountyStateName =paste(CountyName,Postal_Abbr.,sep="") ) %>% 
  select(County=CountyStateName,Permits=8) %>% 
  arrange(desc(Permits)) %>% 
  top_n(20,Permits) -> Top_sf_Permit_Counties

State Level Count

This data set is at the county level. It contains the current level of permits for the particular month by county

data %>% 
  left_join(FipsToState,by=c("FIPSState"="FIPS_Code"))%>% 
  group_by(Postal_Abbr.) %>% 
  summarise(SingleFamilyPermits = sum(Bldgs)) %>% 
  arrange(desc(SingleFamilyPermits))-> State_Counts
knitr::kable(
  State_Counts %>% 
    top_n(10,SingleFamilyPermits),caption = "Top 10 Single Family Permits in April 2023, Statewise"
)
